Detailed Description

template<class T>
class Stack< T >

CMP 246 Module 5 generic stack.

Stack is a generic stack data structure. It supports push, pop, and peek methods, and clear, isEmpty, and size administrative methods. It also has a copy constructor, and overrides the assignment and stream insertion operators.
