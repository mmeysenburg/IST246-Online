Detailed Description

template<class T>
class IteratorSLL< T >::Iterator

IteratorSLL iterator.

This class allows IteratorSLL users to iterate through the list, from front to back, without exposing the pointer structure of the list, and without incurring the time complexity of successive calls to the index-based get() and set() methods.

Initializing constructor.

This constructor makes an iterator that refers to the Node at the end of the specified pointer. The constructor is private, so that the only way to get an iterator is via the IteratorSLL front() and end() methods.
