Detailed Description

CMP Module 3 class representing a movie.

Simple class representing a movie in a movie database. The movie has fields for the title of the movie, the year of its release, and a list of keywords describing the movie. The class has overrides for stream insertion, and the equality operator, which compares a movie object to a keyword string and returns true if the keyword is in the movie's keyword list.
