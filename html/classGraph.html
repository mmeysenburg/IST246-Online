Detailed Description

CMP 246 Module 7 non-templated class representing a weighted graph G=(V, E).

This class uses an adjacency-list representation for the graph. The code is based on the graph algorithms discussed in

Cormen, T. H., &; Leiserson, C. E. (2009). Introduction to algorithms, 3rd Ed.

double *  pDist,
size_t *  pPred 
) const

Perform breadth-first search on this graph.

This method performs a breadth-first search to look find the shortest path, in terms of edge count, from a vertex s to any other vertex in the graph.

Parameters
sVertex to search from
pDistArray of doubles with size |V| + 1. Upon return, array elements [1, ..., |V|] will contain the distance from vertex s to all the other vertices, in terms of number of edges.
pPredArray of size_t values with size |V| + 1. Upon return, array elements [1, ..., |V|] will contain predecessor elements. These allow us to work backwards to create the minimum hop count path from s to any other element in the graph.
Exceptions
std::out_of_rangeif s is < 1 or > |V|.

Perform Dijkstra's algorithm on this graph.
