Detailed Description

template<class T>
class Queue< T >

CMP 246 Module 5 generic queue.

Queue is a generic queue data structure. It supports enqueue and dequeue methods, and clear, isEmpty, and size administrative methods. It also has a copy constructor, and overrides the assignment and stream insertion operators.
